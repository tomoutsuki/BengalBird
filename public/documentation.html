BengalBird — Lesson Authoring Guide

This document describes the JSON data format used by BengalBird to define lessons and exercises. Use it to create new lessons or modify existing ones.

1. File Structure

data/
  chapters.json          ← master index of all chapters, sections & lessons
  lesson1.json           ← exercises for Lesson 1
  lesson2.json           ← exercises for Lesson 2
  romanisation.json      ← Roman–Bengali transliteration mappings
assets/
  audio/
    alphabet/            ← single-letter pronunciation files (অ.mp3, etc.)
    listening/           ← sentence/dialogue audio clips
    sfx/                 ← correct.mp3, wrong.mp3
  illustrations/         ← images for figure_choice exercises

2. chapters.json

Defines the course structure. Each chapter has sections; each section lists lesson IDs that map to data/{lessonId}.json.

{
  "chapters": [
    {
      "id": "ch1",
      "title": { "en": "Chapter 1", "ja": "第1章" },
      "description": { "en": "...", "ja": "..." },
      "sections": [
        {
          "title": { "en": "Section 1", "ja": "..." },
          "lessons": ["lesson1", "lesson2"]
        }
      ]
    }
  ]
}

3. Lesson JSON

Each lesson file (data/lessonN.json) has this top-level shape:

{
  "lesson_id": "lesson1",
  "title": { "en": "...", "ja": "..." },
  "description": { "en": "...", "ja": "..." },
  "est_time_mins": 5,
  "exercises": [ ... ]
}

The exercises array contains objects whose shape depends on the type field.

4. Localized Strings

Any field that holds user-visible text should be an object with en and ja keys:

{ "en": "Hello", "ja": "こんにちは" }

5. Exercise Types

Grammar grammar-sort

User arranges word chips into the correct order.

FieldTypeDescription
typestring"grammar-sort"
questionlocalizedPrompt text
wordsstring[]Bengali words to sort
wordsRomanizedstring[]Matching romanized forms (same order as words)
correctOrderstring[]Words in correct order
hintlocalizedOptional hint
{
  "id": "ex1",
  "type": "grammar-sort",
  "question": { "en": "Rearrange: \"Hello, how are you?\"", "ja": "..." },
  "words": ["আছেন?", "কেমন", "নমস্কার,"],
  "wordsRomanized": ["Achhen?", "Kemon", "Nomoshkar,"],
  "correctOrder": ["নমস্কার,", "কেমন", "আছেন?"],
  "hint": { "en": "Greeting first, then question.", "ja": "..." }
}

Conversation conversation_listening

Audio-driven conversation that reveals lines progressively. May include interactive choice points.

FieldTypeDescription
typestring"conversation_listening"

hintlocalizedOptional hint

Dialogue line:

{
  "speaker": "Rahim",
  "bengali": "নমস্কার!",
  "romanized": "Nomoshkar!",
  "translation": { "en": "Hello!", "ja": "こんにちは！" },
  "audio": "assets/audio/listening/clip.mp3"
}

Choice point:

{
  "type": "choice",
  "question": { "en": "Choose your response:", "ja": "..." },
  "options": [
    { "text": { "bengali": "...", "en": "...", "ja": "..." }, "correct": true },
    { "text": { "bengali": "...", "en": "...", "ja": "..." }, "correct": false }
  ]
}

Figure figure_choice

Image-based word recognition. Displays a Bengali word and a 2×2 image grid; user picks the matching image.

FieldTypeDescription
typestring"figure_choice"
questionlocalizedPrompt
bengalistringBengali word shown prominently
romanizedstringRomanized form
audiostringOptional audio pronunciation path
optionsarrayEach: { image: "path/to/img.svg", label: localized }
correctIndexnumberZero-based index of the correct option
hintlocalizedOptional hint
{
  "id": "ex11_figure_choice",
  "type": "figure_choice",
  "question": { "en": "Which image matches the Bengali word?", "ja": "..." },
  "bengali": "আপেল",
  "romanized": "Apple",
  "options": [
    { "image": "assets/illustrations/apple.svg", "label": { "en": "Apple", "ja": "りんご" } },
    { "image": "assets/illustrations/banana.svg", "label": { "en": "Banana", "ja": "バナナ" } },
    { "image": "assets/illustrations/mango.svg", "label": { "en": "Mango", "ja": "マンゴー" } },
    { "image": "assets/illustrations/orange.svg", "label": { "en": "Orange", "ja": "オレンジ" } }
  ],
  "correctIndex": 0,
  "hint": { "en": "This fruit is red or green.", "ja": "..." }
}

6. romanisation.json

Defines Roman–Bengali transliteration mappings used by the Roman input mode in writing_keyboard exercises.

{
  "mappings": {
    "a": "অ", "aa": "আ", "sh": "শ", "kh": "খ", ...
  },
  "vowelSigns": {
    "আ": "া", "ই": "ি", "ঈ": "ী", ...
  },
  "consonants": ["ক", "খ", "গ", ...],
  "vowels": ["অ", "আ", "ই", ...],
  "hasanta": "্",
  "specialChars": { "0": "০", "1": "১", ".": "।" }
}

How it works: The transliteration engine sorts all mapping keys by length (longest first), then greedily matches the input left-to-right. After a consonant, vowels are automatically converted to their dependent (matra) form using vowelSigns.

7. Adding Audio Files

  • Alphabet: Place MP3 files in assets/audio/alphabet/ named by the Bengali letter, e.g. অ.mp3.
  • Listening: Place audio clips in assets/audio/listening/ with descriptive names, e.g. lesson1_listening1.m4a.
  • SFX: assets/audio/sfx/correct.mp3 and wrong.mp3 are used for feedback sounds.

8. Adding Images

Place illustration files (SVG or PNG recommended) in assets/illustrations/. Reference them in figure_choice exercises by relative path.

9. Adding a New Lesson

  1. Create data/lessonN.json following the structure in Section 3.
  2. Add "lessonN" to the appropriate section in data/chapters.json.
  3. Add any audio files to assets/audio/.
  4. Add any images to assets/illustrations/.
  5. Restart the server and test.
